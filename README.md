# Fizz Buzz

Fizz Buzz kodeoppgave for Epinova

## Getting Started

### Dependencies

* [.Net8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

### Executing program

* Kjøres enkelt via terminalen ved å skrive `dotnet run`
